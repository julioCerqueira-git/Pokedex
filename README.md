# Pokédex React Native

Este projeto é uma **Pokédex** desenvolvida em **React Native** com **Expo** e **TypeScript**, permitindo visualizar informações sobre diversos Pokémon em uma interface simples e estilizada.

---

## Tecnologias Utilizadas

- **React Native**  
- **Expo**  
- **TypeScript**  
- **PokéAPI**

---

## Pré-requisitos

Antes de executar o projeto no seu computador, instale as seguintes ferramentas:

- Node.js (recomendado >= 14)  
- npm ou Yarn  
- Aplicativo Expo Go no celular (opcional, para testar em dispositivo físico)

---

## Como rodar o projeto (desenvolvimento)

1. Clone o repositório:

   git clone https://github.com/julioCerqueira-git/Pokedex.git
   cd Pokedex

2. Instale as dependências:

   Usando npm:
   npm install

   Ou usando Yarn:
   yarn install

3. Inicie o servidor de desenvolvimento do Expo:

   Usando npx:
   npx expo start

4. Abra o app em um emulador ou dispositivo:

   - No terminal do Expo, pressione "a" para abrir no emulador Android (se configurado).
   - Pressione "i" para abrir no simulador iOS (macOS + Xcode).
   - Ou escaneie o QR code com o app Expo Go no seu celular para abrir no dispositivo físico.

---

- Para testar em dispositivo físico, instale o Expo Go (Android/iOS) e escaneie o QR code exibido pelo servidor Expo.
