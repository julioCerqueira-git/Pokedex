# Pokédex React Native

Este projeto é uma **Pokédex** desenvolvida em **React Native** com **Expo**, permitindo visualizar informações sobre diversos Pokémon em uma interface simples e estilizada.

---

##  Tecnologias Utilizadas

- **React Native**  
- **Expo**  
- **TypeScript**  
- **PokéAPI**

---

##  Funcionalidades

- Listagem de Pokémon com imagens e nomes  
- Visualização detalhada de cada Pokémon  
- Design responsivo e intuitivo
